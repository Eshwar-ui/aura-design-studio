# Aura Design Studio — Luxury Home Interior Designer HTML Template

A modern, highly aesthetic, multi-page website template for high-end interior design and architectural studios. Built with pure HTML, Tailwind CSS, and vanilla ES6 JavaScript modules. It features editorial magazine layouts, interactive 3D room hotspot highlights, and a client collaboration portal (Approvals, Timeline tracking, Direct Chat).

## Core Features

- **12 pages built** — Two landing variants, Services, About, Curated Blog, Contact, client & admin dashboards, auth flow, coming soon.
- **Local Typography Assets** — High-end local serifs (`Playfair Display`) and sans (`Inter`) loaded locally via `@font-face` (no CDNs).
- **Tailwind CSS Styling** — Fully responsive utility-based design system conforming to the 4px/8px hierarchy.
- **Premium Interactivity** — Scroll triggers, animated counters, interactive room hotspot detail panels, client approvals, and direct messaging chat simulation.
- **Local Dark Mode** — Clean dark style overrides toggled by class and persisted via `localStorage`.
- **RTL Support** — Seamless direction toggle via navbar and RTL stylesheet adjustments.
- **SEO & Accessibility Ready** — Open Graph tags, Twitter card, JSON-LD structured schema, keyboard navigation, skip links, and ARIA labels.
