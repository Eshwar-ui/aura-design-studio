# Changelog

All notable changes and releases for the Aura Design Studio Luxury Interior Designer template will be documented here.

---

## v1.0.0 — 2026-06-17

### Initial Release

- **12 Pages HTML Structure**: Created full website layouts (Magazine Home, Minimal Home, About, Services, Journal, Contact, client & designer portals, logins, and coming-soon).
- **Design System Implementation**: Integrated Tailwind CSS styling customized around a 4px/8px spacer hierarchy, Alabaster & Brass color palette, and local typography.
- **Local Typography Integration**: Implemented local `@font-face` bindings for Inter and Playfair Display (woff2 files).
- **RTL & Dark Mode Support**: Programmed native class toggles with `localStorage` memory and full RTL coordinate shifts.
- **Interactive Hotspot Room Renderer**: Coded custom hotspots with details popups over living room renders.
- **Interactive Portal Chat**: Created direct chat console between client and designer with automated mock responses.
- **Milestone & Material Tracker**: Implemented status switching (Approve/Decline) inside material selection boards.
- **Form validation**: Created modular validation logic for all forms.
- **SEO & Accessibility Compliance**: Built JSON-LD blocks, OG tags, canonical links, keyboard skip links, and unique meta descriptions on all pages.
